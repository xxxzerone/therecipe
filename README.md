# The Recipe App
TheMealDB 사이트의 api로 레시피 데이터 보여주는 앱

- retrofit으로 api 요청/응답
- converter-gson으로 역직렬화 
- coil-compose 컴포저블로 이미지 출력
- android navigation으로 화면 이동(레시피 상세보기)

## The Meal DB
Free recipe API and database

[TheMealDB](https://www.themealdb.com/)
